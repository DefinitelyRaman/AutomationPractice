Navigation
Ctrl + P: Quick open files by name
Ctrl + Shift + O: Go to symbol in file
Ctrl + Shift + M: Show problems (errors, warnings)
Ctrl + B: Toggle sidebar visibility
Ctrl + Shift + E: Focus on Explorer view

Editing
Ctrl + /: Toggle line comment
Alt + Shift + A: Toggle block comment
Ctrl + D: Select next occurrence of current word
Ctrl + Shift + L: Select all occurrences of current word
Alt + Click: Add multiple cursors

Running & Debugging
F5: Start debugging
Ctrl + F5: Run without debugging
F9: Toggle breakpoint
F10: Step over
F11: Step into

Search & Replace
Ctrl + F: Find in file
Ctrl + H: Replace in file
Ctrl + Shift + F: Find in all files
Ctrl + Shift + H: Replace in all files

Terminal
Ctrl +  ` (backtick): Toggle integrated terminal
Ctrl + Shift +  `: Create new terminal

Formatting & Refactoring
Shift + Alt + F: Format document
Ctrl + K Ctrl + F: Format selected code
F2: Rename symbol
Ctrl + .: Quick fix / refactor
