# AutomationPractice
How to get started
Clone the repository:
git clone https://github.com/DefinitelyRaman/AutomationPractice.git

# Essential GIT commands
git init - initialize a git repository
git add [filename] - make git to track this specific file
touch [filename.exe] - create a file
will be updated...

# Running Cypress Tests
Run Cypress in Debug Mode Use the following command to open the Cypress Test Runner in debug mode:
npx cypress open
Run Cypress in Test Run Mode (Headless) Use the following command to run all Cypress tests in headless mode:
npx cypress run
